# Equitex Lite

## Overview

A Python desktop application that allows users to search stocks, build a portfolio, analyse investments, and visualise market data.

## Features

- Live stock search
- Portfolio management
- Purchase price tracking
- Profit/Loss calculations
- Portfolio analytics
- Historical price charts
- Portfolio allocation charts
- Dark/Light mode
- JSON save/load

## Technologies

- Python
- CustomTkinter
- yfinance
- Matplotlib
- JSON

## How to Run

1. Install requirements.
2. Run:

py run_gui.py
